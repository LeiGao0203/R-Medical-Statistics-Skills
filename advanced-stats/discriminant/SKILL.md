# 判别分析 (Discriminant Analysis)

> 本章内容改编自 阿越就是我 的《R语言实战医学统计》（https://github.com/ayueme/R_medical_stat），采用 CC BY-SA 4.0 许可证发布。

## 适用场景

**什么时候用：**

- 已有明确分类的样本，需要根据多个连续自变量建立模型，对新样本进行**类别预测**
- 需要了解**哪些自变量对分类贡献最大**（判别系数的大小反映区分能力）
- 需要生成判别函数评分（discriminant scores），用于后续可视化或二次分析
- 当分类很确定、自变量与因变量关系较简单时，判别分析可有效替代Logistic回归
- 典型医学场景：根据影像学指标鉴别肿瘤类型（囊肿/胶质瘤/转移瘤）、根据生化指标判断肝硬化分期（早期/晚期）、根据临床症状分类疾病亚型

**什么时候不用：**

- 需要输出OR值、95% CI并有明确的风险因素解读 → 使用Logistic回归
- 自变量和因变量关系非常复杂、非线性强 → 使用决策树或随机森林等机器学习方法
- 仅需将样本按相似度聚成若干类（无已知分类标签） → 使用聚类分析
- 只需要降维而不需要分类判别 → 使用主成分分析或因子分析

## 前置条件

**R包：**

```r
# Fisher判别（线性/二次判别）
install.packages("MASS")
# Bayes判别
install.packages("klaR")
# 可视化
install.packages("ggplot2")
```

**数据格式要求：**

- 因变量（分组变量y）：为分类变量，应为 `factor` 类型，2类或多类均可
- 自变量（x1, x2, x3, ...）：数值型连续变量，用于构建判别函数
- 样本量：每个类别的样本数不宜过少（建议最少5-10例/类），且样本量最好大于自变量个数的3-5倍

**关键统计假设：**

1. **多元正态性**：每个类别中的自变量应近似服从多元正态分布（LDA和Bayes判别都要求）
2. **协方差矩阵齐性**（LDA特有）：各类别的协方差矩阵应当相等。不满足时改用**二次判别分析（QDA）**，QDA允许每类有自己的协方差矩阵
3. **无严重多重共线性**：自变量间不应高度相关，否则判别系数的估计不稳定

## 方法选择决策树

```
你的数据情况 →
├── 需生成线性判别系数、可视化判别函数、简洁解释
│   ├── 各类协方差矩阵相等 → Fisher线性判别（MASS::lda()）
│   └── 各类协方差矩阵不等 → 二次判别分析（MASS::qda()）
│       注意：QDA结果不含判别系数，只有分类结果
├── 需贝叶斯框架下的后验概率、各类均值与标准差 →
│   └── 朴素贝叶斯判别（klaR::NaiveBayes()）
│       适合对概率解释有强需求的场景
└── 分类很确定、结果变量已知、模型解释简单 →
    ├── 优先尝试 lda()（简洁、可解释性强）
    └── 若 lda() 误判较多 → 尝试 qda() 或 NaiveBayes()
```

## 标准工作流

### 步骤1：数据准备与探索

读入数据，确认因变量为 `factor` 类型，自变量为数值型。使用 `str()` 检查变量类型，使用 `table(y)` 查看各类别样本量分布。

```r
df <- read.csv("datasets/例20-1.csv")
table(df$y)
```

### 步骤2：前提条件检验

初步检查各类别的均值和标准差是否有明显差异。可以使用 `by(df[, -c(1, ncol(df))], df$y, colMeans)` 查看各类别中各变量的均值。若需严格检验多元正态性和协方差齐性，可使用 `MVN::mvn()` 或 `heplots::boxM()`。

### 步骤3：执行判别分析

根据方法选择决策树，选择 `lda()`、`qda()` 或 `NaiveBayes()` 拟合模型。公式格式统一为 `y ~ x1 + x2 + x3`。

### 步骤4：评价判别效果

使用 `predict(fit)$class` 获取预测类别，用 `table(实际, 预测)` 构建混淆矩阵，计算正确判别率 = 对角元素之和 / 总样本数。同时查看 `predict(fit)$posterior` 以了解分类的置信度。

### 步骤5：结果报告

论文中应报告：判别函数系数（LDA）、各变量在各类中的均值、先验概率、混淆矩阵及正确判别率。若使用LDA，可报告判别系数的符号和相对大小来说明哪个变量对分类贡献更大。

### Fisher线性判别分析（LDA）

**示例数据：** 孙振球《医学统计学》第4版例20-1，22例肝硬化患者，3个生化指标（x1, x2, x3），y=1为早期（12例），y=2为晚期（10例）。

## 1     1  23   8   0   1
## 2     2  -1   9  -2   1

## 19   19  -9 -20   3   2
## 20   20  -7  -2   3   2
## 21   21  -9   6   0   2

# 拟合LDA模型
fit <- lda(y ~ x1 + x2 + x3, data = df)

输出解读：
- **Prior probabilities**：先验概率，类别1占54.5%，类别2占45.5%

# 可视化判别结果：直方图+密度图
plot(fit, type = "both")
```

上图显示组间判别评分存在一定重合，说明有部分样本可能被错判。

# 提取预测类别并构建混淆矩阵
pred <- predict(fit)$class
table(df$y, pred)

##      1  2
##   1 11  1
##   2  2  8
```

## 结果解读指南

**先验概率（Prior probabilities）：**

各组的样本量占比。如果不指定，默认为各组在训练数据中的比例。先验概率影响后验概率的计算，而预测类别由后验概率的大小决定。

**组均值（Group means）：**

每个自变量在各组中的均值，反映各组的特征轮廓。均值差异越大，该变量对区分类别的贡献通常越大。

**线性判别系数（Coefficients of linear discriminants）：**

LDA的核心输出。构成线性判别函数：LD1 = β1×x1 + β2×x2 + β3×x3。系数的**绝对值大小**反映各自变量对判别评分的相对贡献，系数的**符号**表示方向。例如，x3的系数为0.179（最大），说明x3对分类贡献最大；x2的系数为-0.127（负向），说明x2越高，判别评分越低。

**后验概率（Posterior probabilities）：**

给定自变量观测值后，每个样本属于各类别的概率，所有类别的概率之和为1。概率越大，分类越有把握。接近0.5（二分类时）的概率说明该样本处于分类边界，判别不稳定。

**混淆矩阵（Confusion matrix）：**

行是实际分类，列是预测分类。对角线为正确判别数，非对角为错判数。正确判别率 = 对角线和 / 总和。这个正确判别率是基于训练数据的回代（resubstitution）结果，可能高估真实判别效果。更可靠的做法是使用交叉验证或预留验证集。

**判别评分的可视化：**

`plot(fit, type = "both")` 或 `ggplot2` 直方图可以直观展示判别评分在各类中的分布。如果评分在各类间分布重叠范围大，说明判别效果不佳。

## 常见问题与注意事项

**Q1: LDA中的线性判别系数可以直接比较大小吗？**

可以比较绝对值大小作为相对重要性的参考，但有前提：所有自变量已经在相同尺度上（标准化后）。如果原始变量的单位不同（如年龄vs.血压），系数的大小直接比较不具有意义。可以先使用 `scale()` 对自变量标准化，再拟合LDA。

**Q2: lda() 和 qda() 如何选择？**

关键看各类的协方差矩阵是否相等。如果样本量较少、自变量较多时，LDA更稳健（需要估计的参数少）。QDA每类独立估计协方差矩阵，参数更多，在小样本时容易过拟合。实践中优先尝试LDA，若分类效果不满意再考虑QDA。也可以用 `boxM()` 检验（heplots包）检查协方差齐性。

**Q3: 基于训练数据的正确判别率可信吗？**

回代法获得的正确判别率通常会**高估**真实判别能力（因为模型见过这些数据）。更可靠的方法：
- **留一法交叉验证**：使用 `lda(..., CV = TRUE)`，R将对每个样本逐一留出、用其余样本建模、预测被留出的样本
- **划分训练集/验证集**：用70%的数据建模，30%的数据验证

正确判别率应**在论文中注明是回代结果还是交叉验证结果**。

**Q4: 判别分析与Logistic回归的核心区别是什么？**

Logistic回归输出OR值并提供明确的危险/保护因素解释，更适合病因学研究和风险因素探索。判别分析专注于分类预测和判别评分，更适合诊断模型的建立。此外，Logistic回归对自变量分布的要求更宽松（不需假设多元正态），但要求自变量与logit呈线性关系。

**Q5: 三分类情况下有两组判别系数（LD1和LD2），如何理解？**

对于k类问题，最多可提取k-1个判别函数。LD1是第一个（最重要的）判别函数，解释最多的方差（由Proportion of trace反映）。可视化时，通常用LD1和LD2做散点图（类似主成分分析的PC1和PC2），观察不同类别在二维判别空间的分布。LD1解释绝大多数判别信息时（如iris中99.12%），仅用LD1也可获得较好的分类效果。

**Q6: 朴素贝叶斯判别的“朴素”是什么意思？**

朴素贝叶斯假设各自变量之间**条件独立**（给定类别后自变量不相关），即“每个自变量独立地提供证据，不互相影响”。这个假设在实际数据中往往不成立，但朴素贝叶斯在很多场景仍然表现良好。如果自变量高度相关，结果可能会有偏差，此时更推荐LDA或QDA。

**Q7: 判别分析在医学论文中如何报告？**

建议报告以下内容：
- 判别函数：列出各变量的判别系数（LDA）或各类均值/标准差表（NaiveBayes）
- 混淆矩阵：列出实际分类与预测分类的交叉表
- 正确判别率：写明计算方法（回代法/交叉验证）
- 可视化：判别评分的直方图或散点图（3类以上时可用LD1-LD2散点图）
- 如果做了交叉验证，报告交叉验证的正确判别率
