# 方差分析注意事项 (ANOVA Considerations & Pitfalls)
> 本章内容改编自 阿越就是我 的《R语言实战医学统计》（https://github.com/ayueme/R_medical_stat），采用 CC BY-SA 4.0 许可证发布。

## 适用场景

- **使用本技能的时机**：正在使用或准备使用R语言进行方差分析，需要理解R与SPSS/SAS在ANOVA结果上的差异，需要判断选择何种类型（Type I/II/III）的平方和分解。
- **医学研究场景**：多组均数比较、随机区组设计、协方差分析（ANCOVA）中，当研究设计存在非均衡样本量或多因素时。
- **不适用本技能的情况**：具体某个方差分析的完整操作流程（如单因素ANOVA、重复测量ANOVA、协方差分析等），请使用对应专项技能。

## 前置条件

- **R包要求**：
  ```r
  install.packages("car")  # 提供 Anova() 函数支持 Type II/III
  ```
- **前置知识**：已掌握单因素和多因素方差分析的基本概念，了解 `aov()` 函数的基础用法。
- **统计概念**：理解平方和分解（总平方和 = 组间 + 组内）、均衡设计（各组件本量相等）与非均衡设计的区别。

## 方法选择决策树

```
你的方差分析数据 →
├── 均衡设计（各组件本量相等）
│   ├── 无协变量 → Type I/II/III 结果完全相同，直接用 aov() 即可
│   └── 有协变量（连续型） → Type I 与 Type II/III 结果不同，推荐 Type II 或 Type III
├── 非均衡设计 + 仅一个分组因素 → Type I/II/III 结果相同，直接用 aov()
└── 非均衡设计 + 多个因素 或 存在协变量
    ├── 主效应无交互，关心各因素独立贡献 → Type II（car::Anova(fit, type=2)）
    └── 存在交互项，关心调整后主效应 → Type III（car::Anova(fit, type=3)）
```

> **核心规则**：SPSS/SAS默认使用Type III；R的 `aov()` 默认Type I。当设计非正交时（因子和/或协变量相关），三种类型结果不同，需谨慎选择。

## 标准工作流

### 步骤1：确认研究设计类型

检查数据是否均衡（`table(df$group)` 查看每组件本量），确认是否有协变量（数值型连续变量），确认因素数量。

### 步骤2：选择平方和类型

- 均衡设计（无协变量）→ 直接使用 `aov()`，无需切换类型
- 非均衡设计且仅一个分组因素 → 直接使用 `aov()`
- 非均衡设计且多因素或存在协变量 → 使用 `car::Anova(fit, type=2)` 或 `type=3`

### 步骤3：执行分析并比较（必要时）

当不确定时，同时运行三种类型的结果对比，确保结论一致性。若结果差异大，需要在论文中报告所选类型及理由。

### 步骤4：设定公式中效应顺序（仅Type I重要）

Type I的结果依赖于公式中各项的排列顺序。通用准则：
1. 协变量放在最前
2. 主效应次之
3. 双因素交互项再次之
4. 越基础性的变量越靠前（如性别放在处理方式之前）

### 步骤5：论文报告要点

报告中注明使用的平方和类型（如"采用Type III平方和分解进行方差分析"），说明是否满足方差齐性和正态性假设。

## 结果解读指南

| 输出项 | 含义 | 解读要点 |
|---------|------|----------|
| `Sum Sq` | 各效应对应的平方和 | Type I随公式顺序变化；Type II/III不受公式顺序影响 |
| `Df` | 自由度 | 分组因素自由度 = 组数 - 1 |
| `Mean Sq` | 均方 = Sum Sq / Df | 用于计算F值 |
| `F value` | F统计量 | F = 组间均方 / 残差均方 |
| `Pr(>F)` | p值 | p < 0.05 表示该效应具有统计学意义 |

**Type III额外输出 `(Intercept)` 行**：表示截距项的检验，通常不关心其显著性。

**论文写作示例**：
> "采用III型平方和的两因素方差分析，结果显示A因素的主效应有统计学意义（F(2, 86) = 58.48, p < 0.001），协变量x对因变量有显著影响（F(1, 86) = 177.83, p < 0.001）。"

## 常见问题与注意事项

### Q1: R的 `aov()` 和 SPSS/SAS 结果不一致怎么办？
**A**: R默认使用Type I，SPSS/SAS默认使用Type III。对于均衡设计且无协变量的情况，三者结果一致。不一致时改用 `car::Anova(fit, type = 3)` 即可与SPSS/SAS对齐。

### Q2: 什么时候必须指定平方和类型？
**A**: 当研究设计非均衡且包含多个因素，或者存在协变量时。均衡设计或无协变量的单因素设计下，三者结果完全相同，无需额外指定。

### Q3: Type I中，`aov()` 公式里效应的顺序有影响吗？
**A**: 仅在两种情况下有影响：①因子不止一个且为非平衡设计；②存在协变量。通用做法是协变量最先，主效应次之，交互项最后；越基础性的主效应越靠前。

### Q4: R如何区分协方差分析（ANCOVA）和随机区组ANOVA？
**A**: R通过变量类型自动判断。当 `aov(y ~ x + group)` 中的 `x` 是数值型时，按ANCOVA处理；`x` 是因子型或字符型时，按随机区组ANOVA处理。这是符合《医学统计学》规范的——协变量必须是连续变量，且不能是影响处理的变量。

### Q5: 三种平方和类型可以总结为一句话吗？
**A**: Type I是顺序型（sequential），按公式顺序分配；Type II检验各主效应在控制同阶及以下其他主效应后的贡献；Type III检验各效应在控制所有其他效应后的贡献（SPSS默认）。医学论文建议优先使用Type III以获得与SPSS一致的结果。

### Q6: 医学论文中推荐使用哪种类型？
**A**: 参考SPSS/SAS的默认策略，推荐使用 **Type III**。若研究设计为均衡且无协变量，使用Type I也可（结果一致）。务必在方法部分注明所选平方和类型。
