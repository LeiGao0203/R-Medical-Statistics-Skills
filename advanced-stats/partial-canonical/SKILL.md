# 偏相关和典型相关分析 (Partial and Canonical Correlation Analysis)

> 本章内容改编自 阿越就是我 的《R语言实战医学统计》（https://github.com/ayueme/R_medical_stat），采用 CC BY-SA 4.0 许可证发布。

## 适用场景

**偏相关分析**使用场景：
- 在临床研究中，需要排除混杂因素（如年龄、性别）后评估两个变量的真实相关性
- 当存在第三个变量同时影响两个研究变量时，需计算净相关关系
- 例：控制肺活量后研究身高与体重的相关性

**典型相关分析**使用场景：
- 研究一组自变量与一组因变量之间的整体相关性
- 探索生长发育指标（身高、体重、胸围、肺活量）与身体素质指标（短跑、跳高、跳远、实心球）的综合关系
- 当单个因变量无法全面反映结局时的多变量关联研究

**不使用本方法的情况：**
- 仅分析两个变量的简单线性相关 → 使用 Pearson/Spearman 相关分析
- 一个因变量多个自变量的预测建模 → 使用多元线性回归
- 降维或变量简化 → 使用主成分分析或因子分析
- 多个自变量与一个因变量的关系 → 使用多元回归而非典型相关

## 前置条件

**R包安装：**

```r
install.packages("ppcor")   # 偏相关分析
install.packages("CCP")     # 典型相关显著性检验
install.packages("haven")   # 读取SPSS数据文件
```

**偏相关分析要求：**
- Pearson偏相关：变量需为连续型数值变量，近似服从正态分布
- Spearman偏相关：变量可为有序分类或非正态连续变量
- 控制变量应为连续型或可量化的变量
- 数据中不应有缺失值

**典型相关分析要求：**
- 两组变量均应为连续型数值变量
- 样本量应大于两组变量数之和（n > p + q）
- 变量之间应存在一定的线性相关性
- 两组变量内部不应有高度共线性

## 方法选择决策树

```
你的数据情况 →
├── 两个连续变量，需控制第三个变量
│   ├── 变量近似正态分布 → Pearson偏相关 (method="pearson")
│   └── 变量为非正态或含有序分类 → Spearman偏相关 (method="spearman")
├── 两组多变量的整体关联分析
│   └── 两组均为连续变量 → 典型相关分析 (cancor + CCP检验)
├── 仅两个变量的简单相关（不控制第三变量）→ 不用本方法，用cor.test()
└── 一个因变量 + 多个自变量 → 不用本方法，用多元回归
```

## 标准工作流

### 步骤1：数据准备与探索

加载数据，确认变量结构。偏相关分析需明确：哪两个是研究变量，哪个是控制变量。典型相关分析需将变量分为两组。

```r
# 偏相关数据
library(ppcor)
df <- haven::read_sav("datasets/data01.sav")
df1 <- df[, 2:4]
names(df1) <- c("height", "weight", "vc")  # vc是需要控制的因素
head(df1)

# 典型相关数据（孙振球《医学统计学》第四版，例23-1）
df <- read.csv("datasets/例23-1.csv", header = T)
head(df)
# 前4列：生长发育指标（肺活量、身高、体重、胸围）
# 后4列：身体素质指标（短跑、跳高、跳远、实心球）
```

### 步骤2：前提条件检验

- 偏相关（Pearson）：检查变量是否近似正态，可用 `shapiro.test()` 或查看直方图/Q-Q图
- 偏相关（Spearman）：确认变量为有序或非正态
- 典型相关：检查样本量是否充足（n > p + q），检查两组变量内部是否存在多重共线性

### 步骤3：执行统计分析

```r
# Pearson偏相关：height和weight的相关性，控制vc
p1 <- pcor(df1, method = "pearson")
p1
# 结果包含：$estimate（偏相关系数矩阵）、$p.value、$statistic、$n

# Spearman偏相关：x和y的相关性，控制z
df2 <- haven::read_sav("datasets/data02.sav")
names(df2) <- c("id", "x", "y", "z")
pcor(df2[, -1], method = "spearman")

# 典型相关分析
cc1 <- cancor(df[, 1:4], df[, 5:8])
cc1
# 结果包含：$cor（典型相关系数）、$xcoef、$ycoef、$xcenter、$ycenter

# 典型相关显著性检验
library(CCP)
rho <- cc1$cor
n <- dim(df[, 1:4])[1]
p <- length(df[, 1:4])
q <- length(df[, 5:8])
p.asym(rho, n, p, q, tstat = "Wilks")
```

### 步骤4：结果解读

偏相关结果中：
- `$estimate[1,2]`（即height和weight交叉位置）为偏相关系数：0.794
- `$p.value[1,2]`：p < 0.001，表示控制vc后height和weight的相关性有统计学意义
- `$statistic`：t检验统计量

典型相关结果中：
- `$cor` 给出4个典型相关系数：[0.886, 0.279, 0.194, 0.038]
- `p.asym()` 的Wilks检验显示：仅第一个典型相关系数p < 0.001，后续3个均不显著（p > 0.05）
- 结论：生长发育指标与身体素质指标之间存在一个有统计学意义的典型变量对

### 步骤5：结果报告

论文中报告格式示例：

> 偏相关分析显示，在控制肺活量后，身高与体重之间的偏相关系数为0.794（t=5.39，p<0.001），表明排除肺活量影响后身高与体重仍呈显著正相关。

> 典型相关分析显示，生长发育指标与身体素质指标的第一个典型相关系数为0.886（Wilks' λ=0.191，F=10.48，p<0.001），后三个典型相关系数均无统计学意义。

### 示例3：偏相关散点图

偏相关散点图通过分别计算两个变量对控制变量的回归残差来绘制：

```r
# 计算height ~ vc 的残差
residX <- resid(lm(height ~ vc, data = df1))
# 计算weight ~ vc 的残差
residY <- resid(lm(weight ~ vc, data = df1))

# 两残差的相关系数即为偏相关系数
cor(residX, residY, method = "pearson")
## [1] 0.7941292

# 将残差加上各自均值使坐标轴可解释
residX1 <- residX + mean(df1$height)
residY1 <- residY + mean(df1$weight)
plot(residX1, residY1, xlab = "身高", ylab = "体重")
```

## 结果解读指南

**偏相关系数（$estimate）：**
- 矩阵对角线为1（变量与自身的偏相关）
- 非对角线元素为控制其他变量后的偏相关系数，取值-1到1
- height和weight的偏相关系数为0.794，表示在控制vc后二者呈高度正相关

**偏相关的p值（$p.value）：**
- 对角线为0（自身相关无检验值）
- height和weight间的偏相关p < 0.001，具有显著统计学意义
- height和vc间的偏相关p = 0.406，无统计学意义

**偏相关的检验统计量（$statistic）：**
- 为t检验统计量，绝对值越大相关性越强
- height-weight: t = 5.39，对应自由度n-3=17

**典型相关系数（$cor）：**
- 输出个数 = min(p, q)，本例4个
- 第1典型相关系数0.886，表示两组变量存在较强的线性综合关系
- 第2-4相关系数依次降低

**典型相关显著性检验：**
- `p.asym()` 默认Wilks' Lambda，还支持Hotelling、Pillai、Roy
- "1 to 4"行：检验所有4个典型相关系数是否全为零，p<0.001说明至少第1个显著
- "2 to 4"行：检验第2-4个是否全为零，p=0.393说明第2个起均不显著
- 最终结论：仅第一对典型变量有意义

**非标准系数的解释（$xcoef / $ycoef）：**
- 这些是原始尺度上的权重系数，用于计算典型变量得分
- 系数绝对值的相对大小反映各变量对典型变量的贡献
- 注意：这些系数受变量尺度影响，解释时应结合标准化系数

## 常见问题与注意事项

**Q1: 偏相关和简单相关有何区别？**
偏相关剔除了控制变量的线性影响后再计算两个变量的相关性，反映的是“净相关”。简单相关可能混杂了第三变量的间接效应。如果控制变量后相关系数发生明显变化，说明存在混杂效应。

**Q2: 什么时候用Pearson偏相关，什么时候用Spearman？**
变量为连续型且近似正态分布时用Pearson；变量含有序分类（如等级数据“轻/中/重”）或不服从正态分布时用Spearman。Spearman基于秩次计算，对异常值不敏感。

**Q3: 偏相关散点图中坐标轴为残差，如何让坐标轴可解释？**
将残差分别加上对应变量的均值（`resid + mean(x)`），这样散点图的形状不变，但坐标轴数值恢复为原始量纲范围。

**Q4: 典型相关输出多个相关系数，如何取舍？**
通过 `p.asym()` 的Wilks检验逐层判断：先检验所有系数是否全为零；若p<0.05再依次检验剩余系数。典型做法是只保留第一个显著的典型变量对或前几个显著的结果。

**Q5: cancor()返回的$xcoef和$ycoef是标准化系数吗？**
不是。`cancor()` 返回的是原始尺度上的非标准化系数。若需要标准化系数，可先对数据做 `scale()` 标准化后再调用 `cancor()`。

**Q6: 典型相关分析与多元回归有何不同？**
多元回归有单一因变量（Y），研究多个X对Y的预测关系。典型相关则是X组（多个变量）与Y组（多个变量）之间的双向关联，不区分自变量和因变量。

**Q7: SPSS与R的偏相关结果一致吗？**
一致。偏相关系数的计算公式为确定的数学公式，不同软件结果相同。注意SPSS的偏相关菜单中需指定“控制变量”，R的`pcor()`则对所有变量两两计算控制其余变量的偏相关。

**Q8: 样本量要求？**
偏相关分析中，n应至少大于变量数+10。典型相关分析要求n > p + q，且建议每组变量数不超过5个，否则典型相关系数可能虚高（过拟合）。
