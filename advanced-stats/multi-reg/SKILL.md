# 多元线性回归 (Multiple Linear Regression)

> 本章内容改编自 阿越就是我 的《R语言实战医学统计》（https://github.com/ayueme/R_medical_stat），采用 CC BY-SA 4.0 许可证发布。

## 适用场景

**使用多元线性回归的典型场景：**

- 探索多个自变量（危险因素、生化指标、人口学特征等）对一个连续型因变量的影响方向和大小
- 在控制其他混杂因素的前提下，评估某一特定因素的独立效应
- 建立预测模型，用多个预测变量共同估计连续型结局值
- 分析变量间的数量依存关系及相对作用大小（比较标准化回归系数）

**不使用多元线性回归的情况：**

| 你的情况 | 应使用的方法 |
|----------|-------------|
| 因变量为二分类（如患病/未患病） | Logistic 回归（`glm(y ~ ., family = binomial)`） |
| 结局变量为生存时间（含删失） | Cox 比例风险回归（`survival::coxph()`） |
| 仅一个自变量 | 简单线性回归（`lm(y ~ x)`） |
| 结局为计数数据（如发病次数） | 泊松回归或负二项回归 |
| 自变量间共线性严重且需降维 | 主成分回归、岭回归、LASSO |
| 数据不满足线性假设且无法转换 | 广义可加模型（GAM）、样条回归 |

**医学研究常见应用：**

- 研究空腹血糖受总胆固醇、甘油三酯、胰岛素、糖化血红蛋白等多种生化指标的影响（课本例15-1）
- 探索影响血压水平的多个生活方式和生化因素
- 建立术后住院天数的预测模型

## 前置条件

**R 包安装：**

```r
install.packages(c("GGally", "car", "gvlma", "MASS", "leaps", "performance"))
```

**数据格式要求：**

- 因变量 `y`：连续型数值变量（如血糖值、血压值）
- 自变量 `x1, x2, ..., xk`：连续型数值变量或恰当的虚拟变量（因子型需先哑变量编码）
- 数据框格式，每行一个观测，无缺失值（或已处理缺失值）

**统计假设（LINE 四大条件）：**

1. **线性 (Linearity)**：各自变量与因变量之间存在线性关系，可通过成分残差图（`car::crPlots()`）判断
2. **独立性 (Independence)**：各观测相互独立，使用 Durbin-Watson 检验（`car::durbinWatsonTest()`）
3. **正态性 (Normality)**：残差服从正态分布，使用 Q-Q 图或 Shapiro-Wilk 检验
4. **等方差性 (Equal variance)**：残差方差恒定（同方差性），使用残差拟合图或 `car::ncvTest()`

**额外条件：**

- **无多重共线性**：各自变量间不存在高度线性相关，VIF（方差膨胀因子）通常应 < 5 或 < 10
- **样本量足够**：经验法则为样本量至少为自变量个数的 10–20 倍

## 方法选择决策树

```
你的数据情况 →
├── 有一个连续型因变量 + 多个连续型自变量
│   ├── 重在解释各因素的作用大小 → lm(y ~ x1 + x2 + ...) + summary()
│   ├── 重在预测并比较模型 → 结合 AIC/BIC/调整 R² 选择最优子集
│   └── 自变量较多想简化模型 →
│       ├── 有专业先验知识 → 指定强制纳入变量 + 逐步法
│       ├── 探索性筛选 → 逐步回归（stepAIC direction="both"）
│       ├── 比较所有可能子集 → 全子集回归（leaps::regsubsets）
│       └── 处理多重共线性 → VIF 诊断后剔除高共线性变量
│
├── 自变量含分类变量（多组别）
│   └── 需先将分类变量转换为哑变量 → 多元线性回归等价于协方差分析
│
├── 需诊断模型假设 →
│   ├── 快速图形诊断 → plot(f) 或 performance::check_model(f)
│   ├── 综合统计检验 → gvlma::gvlma(f)
│   └── 单项检验 → car::qqPlot / ncvTest / durbinWatsonTest
│
└── 自变量间有交互作用 → 公式中纳入交互项 x1:x2
```

## 标准工作流

### 步骤 1：数据准备与探索

使用 `str()` 确认数据结构，用 `GGally::ggpairs()` 快速绘制散点图矩阵，观察变量间的两两关系和相关系数。

```r
library(GGally)
ggpairs(df) + theme_bw()
```

从图中可初步判断各自变量与因变量的相关性强弱，以及自变量之间是否存在共线性线索。

### 步骤 2：建立多元线性回归模型

```r
f <- lm(fpg ~ cho + tg + ri + hba, data = df15_1)
summary(f)
```

### 步骤 3：模型评价

通过 R²、调整 R²、AIC、BIC、RMSE 等指标评价模型拟合优度。

### 步骤 4：回归诊断

逐一检验线性、独立性、正态性、等方差性和多重共线性假设。

### 步骤 5：变量筛选与模型优化

根据研究目的选择逐步回归（AIC 准则）或全子集回归（Cp 准则或调整 R²）。

### 步骤 6：结果报告

报告最终模型的回归系数、标准误、各变量的 p 值，以及模型的 R² 和 F 检验结果。

论文描述模板：
> 以空腹血糖为因变量，以总胆固醇、甘油三酯、胰岛素、糖化血红蛋白为自变量，建立多元线性回归模型。结果显示，糖化血红蛋白（β = 0.638，p = 0.016）和胰岛素（β = -0.271，p = 0.036）是空腹血糖的独立影响因素。模型的决定系数 R² = 0.601，整体 F 检验 F(4, 22) = 8.278，p < 0.001。

## 结果解读指南

### summary() 输出解读

| 输出项 | 含义 | 解读要点 |
|--------|------|---------|
| `Residuals` | 残差的极小/极大值和四分位数 | 大致对称且中位数接近 0 为佳 |
| `(Intercept)` | 截距 β₀ | 所有自变量为 0 时的预测值（多数情况下有临床意义解释） |
| `Estimate` | 偏回归系数 βᵢ | 控制其他变量不变时，xᵢ 每增加 1 单位，y 平均改变 βᵢ 单位 |
| `Std. Error` | 回归系数的标准误 | 越小说明系数估计越精确 |
| `t value` | t 检验统计量 = Estimate / Std. Error | 检验该回归系数是否为 0 |
| `Pr(>|t|)` | p 值 | p < 0.05 表示该自变量对因变量有显著贡献 |
| `Residual standard error` | 残差标准误 σ̂ | 模型预测的典型误差大小 |
| `Multiple R-squared` | 决定系数 R² | 回归平方和 / 总平方和，0–1 间，越大说明模型解释的变异越多 |
| `Adjusted R-squared` | 调整 R² | 对模型中自变量个数做惩罚，比 R² 更适合比较不同自变量数的模型 |
| `F-statistic` | 整体模型 F 检验 | 检验所有自变量联合是否显著，p < 0.05 表示模型整体有意义 |

### 回归系数的解读

- 偏回归系数 βᵢ 表示「在控制其他自变量的情况下，xᵢ 每变化 1 个单位对 y 的独立效应」
- 系数的正负号表示作用方向：正号 = y 随 x 增大而增大；负号 = y 随 x 增大而减小
- 若需比较不同量纲变量的相对作用大小，可使用标准化回归系数（对自变量和因变量均做标准化后再回归）

### 变量筛选结果解读

- **AIC（赤池信息准则）**：越小越好，stepAIC() 每一步剔除或纳入都是为了使 AIC 降低
- **Cp 统计量**：Mallows' Cp，越接近模型中包含的参数个数说明模型越好
- 后退法从全模型开始逐步剔除不显著变量；前进法则逐步纳入显著变量；逐步回归两者结合

### 模型诊断图解读

| 图形 | 判断标准 | 不满足时的对策 |
|------|---------|---------------|
| 残差拟合图（左上） | 点随机分布在水平线（y=0）两侧 | 考虑变量变换、添加二次项或非线性方法 |
| 正态 Q-Q 图（右上） | 点基本沿对角线分布 | 因变量变换（如 log 变换）或使用稳健回归 |
| 位置尺度图（左下） | 水平线两侧的点随机分布 | 使用加权最小二乘法或变量变换 |
| 残差杠杆图（右下） | 无点出现在库克距离等高线外 | 检查原始数据，考虑排除或做敏感性分析 |

## 常见问题与注意事项

**Q1: R² 多大才算好？调整 R² 和 R² 有什么区别？**

A: R² 没有绝对标准，取决于研究领域。医学数据中 0.3–0.7 都常见，探索性研究侧重解释变量作用而非追求高 R²。R² 随自变量增加而单调递增（即使变量无意义），调整 R² 会对自变量个数做惩罚：当新加入的变量贡献有限时，调整 R² 可能反而下降。比较不同模型时应使用调整 R²。

**Q2: 逐步回归 (`stepAIC`) 的 direction 参数选什么？三者有何区别？**

A:
- `"backward"`（后退法）：从全模型开始，逐步剔除最无用的变量。至少保留部分理论上重要的变量。
- `"forward"`（前进法）：从空模型开始，逐步纳入最有用的变量。初始模型 AIC 最低但可能错过交互效应。
- `"both"`（逐步回归）：每一步同时考虑加入或剔除变量，最为常用。自动决策但结果需结合临床意义判断。

**注意**：逐步选择法给出的「最优模型」只是基于 AIC 的统计最优，不代表临床最优。建议：
- 将专业上公认的重要变量强制纳入模型
- 对逐步法筛选结果做敏感性分析（不同方向是否一致）
- 最好结合全子集回归（`regsubsets`）交叉验证

**Q3: VIF 多大表示存在严重的多重共线性？如何处理？**

A: 经验规则：
- VIF < 5：基本不存在共线性问题
- 5 ≤ VIF < 10：存在一定共线性
- VIF ≥ 10：存在严重共线性，需处理

处理策略（按优先级）：(1) 剔除共线性严重的变量中临床意义较小者；(2) 合并高度相关的变量为复合指标；(3) 使用岭回归、LASSO 或主成分回归。

**Q4: gvlma 结果显示 Link Function NOT satisfied 是什么意思？**

A: Link Function 检验检测因变量与预测变量的函数形式是否合适。p < 0.05 提示可能不是简单线性关系，但 `lm()` 的 link 默认是 identity（即线性连接），这个警告通常不影响使用。若伴随其他假设也不满足，可考虑对因变量做变换（log、平方根等）或使用广义可加模型（GAM）。

**Q5: `summary()` 中整体 F 检验显著但个别变量不显著，是怎么回事？**

A: 可能的原因：
- 不显著变量和显著变量间存在共线性，导致单个系数的 t 检验不显著但整体模型显著
- 实际样本量偏小，个别变量效应无法检出
- 对策：检查 VIF，若有共线性则做变量筛选

**Q6: 分类自变量如何处理才能纳入多元线性回归？**

A: 必须转换为哑变量（dummy variable）。二元分类变量可直接用 0/1，多分类变量（k 类）需设置 k-1 个哑变量，回归系数表示该类别相对于参照组的效应。R 中若变量为 factor 型，`lm()` 会自动创建哑变量；也可使用 `model.matrix()` 手动构造。

**Q7: 样本量至少需要多少？**

A: 经验法则：
- 最小要求：n ≥ k + 5（k = 自变量个数），但仅能求出参数，精度很差
- 一般建议：n ≥ 10k 或 n ≥ 15k（10–15 个样本/变量）
- 若需用逐步回归等自动筛选方法，建议 n ≥ 20k
- 本例中 n = 27，k = 4，符合 10k = 40 的经验要求偏紧，解释时应注意效度问题
