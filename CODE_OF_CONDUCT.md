# Code of Conduct

本项目希望保持友善、专业、开放的协作氛围。

## Expected Behavior

- 尊重不同背景和经验水平的贡献者。
- 讨论问题时聚焦事实、代码、统计方法和文档本身。
- 对医学统计内容保持谨慎，欢迎指出错误并提供可验证依据。
- 接受建设性反馈，也以建设性的方式提出反馈。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视性语言或持续的冒犯行为。
- 明知有误仍传播可能造成严重误导的医学或统计建议。
- 在讨论中公开他人隐私信息。

## Reporting

如果你发现不当行为，可以通过 GitHub issue 或维护者公开提供的联系方式报告。维护者会根据情况处理，包括提醒、删除不当内容或限制参与。
