# Contributing

感谢你考虑贡献这个项目。

## Scope

适合提交的内容包括：

- 新增医学统计方法 skill。
- 修正已有 skill 中的统计解释、R 代码或方法选择逻辑。
- 增加可复现的 notebook、示例数据说明或案例。
- 改进 Jupyter Notebook skill 的模板、脚本或质量检查流程。

## Skill Guidelines

- 每个 skill 必须包含 `SKILL.md`。
- `SKILL.md` 的 frontmatter 必须包含 `name` 和 `description`。
- 统计类 skill 应说明适用场景、不适用场景、前提条件、R 包依赖、标准工作流、结果解释和常见错误。
- 代码示例应尽量可直接运行，并避免依赖本仓库未提供的数据文件。
- 如果内容改编自外部资料，请在 skill 中保留清晰署名、来源链接和许可证说明。
